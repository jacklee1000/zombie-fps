# zombie-fps — Image Assets

A collection of concept/asset art for a zombie-themed first-person shooter (FPS):
enemy characters, mutant bosses, environments, and weapons. All assets are static
image files (`.png` / `.jpg`) intended as reference art or sprite/UI source material.

Most art was generated with the **Lucid Origin** model (filenames prefixed
`lucid-origin_`); the weapon renders were produced with **ChatGPT** and **Copilot**
image tools.
